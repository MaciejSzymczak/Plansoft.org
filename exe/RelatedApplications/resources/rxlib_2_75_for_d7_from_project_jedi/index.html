RxLib 2.75 for D7 (from Project JEDI) (ID #18589)


Project JEDI

The official D6 version of RxLib 2.75 converted to Delphi 7 by Project JEDI. It contains also "JVCL Convert" to convert RxLib-based projects to JEDI-VCL.

RxLib contains several Freeware VCL components. They are currently incorporated into JVCL, but for those who still want to use the original components, this conversion should help to move to Delphi 7.
